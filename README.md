# Cyber Defense & Threat Modeling

This repository contains structured cybersecurity analyses focused on threat modeling, incident response reasoning, human-factor risk, and layered defense strategies.

The work demonstrates practical application of:

- CIA Triad evaluation
- Threat modeling frameworks
- Breach analysis and adversarial thinking
- Risk mitigation planning
- Layered security architecture (SIEM, NIDS/NIPS, VPN)
- Cryptographic tradeoff analysis (RSA vs AES)
- Policy and human-factor security controls

## Focus Areas

### Threat Modeling
Comparative analysis of enterprise breach scenarios including:
- Retail payment system compromise
- Nation-state intrusion campaigns
- Federal personnel data breach

Each model evaluates:
- Attack vectors
- Vulnerabilities
- Unauthorized outcomes
- Defensive improvements

### Risk & Defense Strategy
Includes layered defense recommendations integrating:
- Security awareness training
- Access control and least privilege
- Behavior analytics and monitoring
- VPN and encryption strategy evaluation

## Objective

The goal of this repository is to demonstrate structured security analysis and risk-based decision-making aligned with modern cybersecurity defense practices.
